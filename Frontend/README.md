# Smarter Foster Management for Safer Animal Care

## Overview

A web-based management interface that allows employees to view, create, edit, and delete foster homes within the system. This tool helps streamline foster home administration by providing a centralized dashboard for managing home information.

### About the Project

This project was developed during a 6-week cohor(Feb-March 2026). A team of developers who created Pet Foster Project where users can adopt and foster animals. Users have the option to buy products for their pets. Employees have the option approve pet requests for pet parents

---

# Foster Homes Management System

A web-based management interface that allows employees to view, create, edit, and delete foster homes within the system. This tool helps streamline foster home administration by providing a centralized dashboard for managing home information.

---

## Features

- View all foster homes in a table
- Add new foster homes
- Edit existing foster homes
- Delete foster homes
- Responsive user interface
- Integration with backend API
